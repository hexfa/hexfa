## About Me

I’m a **Senior Android & Flutter Developer** with **12+ years of experience** building high-performance, secure, scalable, and user-focused mobile applications.

My main focus is not only building beautiful apps, but also designing strong foundations behind them: **mobile architecture, networking, security, API communication, performance, and maintainable code**.

I’m also actively working with **self-hosted AI models and local AI tools** to build private, flexible, and production-friendly AI-powered workflows.

- Building modern **Android & Flutter applications**
- Focused on **Security, Networking, AppSec, API security, and secure mobile architecture**
- Working with **self-hosted AI**, **local LLMs**, **RAG**, **vector search**, and private AI assistants
- Experienced with **Kotlin, Java, Dart, Flutter, Firebase, REST APIs, WebSocket, Linux, Git, and backend integrations**
- Passionate about clean architecture, performance, automation, privacy, and smart mobile products
- Ask me about **Android, Kotlin, Java, Flutter, Web, Network, Security, Firebase, and Self-hosted AI**

> [!IMPORTANT]
> I care deeply about building apps that are not only beautiful and fast, but also secure, reliable, scalable, and privacy-aware.
